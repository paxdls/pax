Welcome to the Pax DLS

How to use:

Downloading a file

To download a file on Pax, simply use a link like this: http://dls.pax.us.to/download/filename

Viewing/Rendering a file

To view/render a file on Pax, simply use a link like this: http://dls.pax.us.to/files/filename.extension

NOTE: Some files can't be viewed. If they can't be viewed then they will be downloaded like with http://dls.pax.us.to/download/...
